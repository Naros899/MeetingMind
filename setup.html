Setup Guide

Follow these steps to get Chronics.ai working. It takes about 5 minutes.

Step A — OpenAI API Key
1 Create an OpenAI account
Go to platform.openai.com/signup and sign up (or log in if you already have an account).
2 Generate an API key
Go to platform.openai.com/api-keys, click Create new secret key, give it a name like "Chronics", and copy the key. It starts with sk-.
3 Add billing
Go to Billing settings and add a payment method. API usage is pay-as-you-go. For 30 meetings, expect roughly $1–3 total.
Tip: Set a spending limit of $10/month in billing settings so there are no surprises.
Step B — EmailJS Setup
1 Create an EmailJS account
Go to emailjs.com and sign up for a free account. The free plan includes 200 emails/month — more than enough for the event.
2 Connect your email service
In the EmailJS dashboard, go to Email Services and click Add New Service. Choose Gmail (or Outlook). Click Connect Account and authorize access. Note down the Service ID (looks like service_xxxxxxx).
3 Create an email template
Go to Email Templates and click Create New Template. Set it up like this:
Subject: {{subject}} To: {{to_email}} {{body}}
Use these template variables:
VariableWhat it does
{{to_email}}Recipient's email address
{{from_name}}Your name (the sender)
{{subject}}Email subject line
{{body}}The full email body
In the template settings, set To Email to {{to_email}} and From Name to {{from_name}}. Save the template. Note down the Template ID (looks like template_xxxxxxx).
4 Get your Public Key
Go to Account (top right menu) and find your Public Key under the API Keys section. Copy it.
Tip: Make sure the "To Email" field in your template settings is set to {{to_email}} — not a hardcoded address. This is what allows sending to different people.
Step C — Enter Settings in Chronics.ai
1 Open the app and expand Settings
Open Chronics.ai and tap the Settings card at the top.
2 Paste your 4 values
Enter the following:
OpenAI API Key: sk-... EmailJS Service ID: service_xxxxxxx EmailJS Template ID: template_xxxxxxx EmailJS Public Key: your_public_key
Click Save Settings. You'll see a "Configured" badge. These are saved locally and you won't need to enter them again on this device.

You're all set

Your settings are saved. Start using Chronics.ai to capture meetings and send follow-ups instantly.
